\ccDefinition
  
The concept \ccRefName\ describes the types of surfaces to be meshed.
The surface types 
are    required to be copy constructible
and assignable.

%\ccRefines

%ThisConcept \\
%ThatConcept

\ccTypes

In addition, surface types are required
\begin{itemize}
\item 
either to provide a nested type:
\ccNestedType{Surface_mesher_traits_3}{a model of \ccc{SurfaceMesherTraits_3}}
%\ccc{Surface::Surface_mesher_traits_3}
%that is a model of \ccc{SurfaceMeshTraits_3}
\item or to be a surface type for which a specialization
of the traits generator \ccc{Surface_mesh_traits_generator_3<Surface>}
exists.
\end{itemize}
