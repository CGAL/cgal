\ccDefinition  
The class \ccRefName\ is a model of the concept
\ccc{RegularTriangulationVertexBase_2}. It is the default
vertex base class of regular triangulations.


\ccInclude{CGAL/Regular_triangulation_vertex_base_2.h}

\ccParameters
The template parameters \ccc{Traits} has to be a model
of \ccc{RegularTriangulationTraits_2}.

The template parameters \ccc{Vb} has to be a model
of the concept \ccc{TriangulationVertexBase_2} and is by default
instantiated by \ccc{CGAL::Triangulation_vertex_base_2<Traits>}.
