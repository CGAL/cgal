\ccDefinition

The template parameter \ccc{StreamLinesTraits_2} has to be
instantiated by a model of the concept \ccc{StreamLinesTraits_2}.\\

This class provides a vector field specified by a set of sample points
defined on a triangulated domain. All sample points are inserted to a
\ccc{Delaunay triangulation}, and for each point \ccc{p} in the domain
located in a face \ccc{f}, its vector value is interpolated from the
vertices of the face \ccc{f}.
