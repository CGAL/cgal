\ccDefinition
  
The class \ccRefName\ is a model of the concept
\ccc{SurfaceMeshCellBase_3}.
It is designed to serve  as vertex base  class
in a  triangulation class \ccc{Tr} 
plugged in a  \ccc{Surface_mesh_complex_2_in_triangulation_3<Tr>}
class.

The first template parameter is the geometric traits class.

The second template parameter is a base class. It
has to be a model of the concept \ccc{TriangulationCellBase_3}
and  
defaults to \ccc{Triangulation_cell_base_3 <GT>}.
