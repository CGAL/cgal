\begin{ccRefClass}[Nef_polyhedron_3<Traits>::]{SFace_cycle_iterator}

\ccDefinition

The type \ccc{SFace_cycle_iterator} iterates over a list of 
\ccc{Object_handles}. Each item of that list can either be assigned
to \ccc{SVertex_handle}, \ccc{SHalfedge_handle} or \ccc{SHalfloop_handle}. 
To find out which
of these assignment works out, the member functions \ccc{is_svertex()}, 
\ccc{is_shalfedge()} and \ccc{is_shalfloop()} are provided.
