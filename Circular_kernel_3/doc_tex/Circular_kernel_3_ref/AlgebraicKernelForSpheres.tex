\begin{ccRefConcept}{AlgebraicKernelForSpheres} 

\ccDefinition

The \ccc{AlgebraicKernelForSpheres} concept is meant to provide the
curved kernel with all the algebraic functionalities required for the
manipulation of spheres, circles, and circular arcs in 3D. 

\ccHasModels

\ccc{Algebraic_kernel_for_spheres_2_3}
