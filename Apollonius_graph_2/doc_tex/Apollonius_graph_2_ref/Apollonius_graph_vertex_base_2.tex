\ccDefinition

The class \ccRefName\ provides a model for the
\ccc{ApolloniusGraphVertexBase_2} concept which is the vertex base
required by the \ccc{ApolloniusGraphDataStructure_2} concept. The
class \ccRefName\ has two template arguments, the first being the
geometric traits of the Apollonius graph and should be a model of the
concept \ccc{ApolloniusGraphTraits_2}. The second is a Boolean which
controls whether hidden sites are actually stored. Such a
control is important if the user is not interested in hidden sites
and/or if only insertions are made, in which case no hidden
site can become visible. If \ccc{StoreHidden} is set to
\ccc{true}, hidden sites are stored, otherwise they are
discarded. By default \ccc{StoreHidden} is set to \ccc{true}.
