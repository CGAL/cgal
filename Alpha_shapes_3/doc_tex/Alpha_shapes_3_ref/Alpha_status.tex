\ccDefinition
  
The class \ccRefName\ is a small data structure to store
the critical alpha values of faces of an alpha shape.
Each face has three critical alpha values, called
\ccc{alpha_min}, \ccc{alpha_mid} and \ccc{alpha_max} in increasing order.
The face will be exterior for any $\alpha < $ \ccc{alpha_min},
singular for \ccc{alpha_min} $\leq  \alpha  < $ \ccc{alpha_mid},
regular for  \ccc{alpha_mid} $\leq  \alpha < $ \ccc{alpha_max}
and interior for \ccc{alpha_max} $ \leq alpha$.
The value \ccc{alpha_min} is undefined for faces which are not Gabriel
faces and therefore do not appear in the alpha complex
without any of their
including face. The value \ccc{alpha_max} is undefined
for convex hull faces which can never be interior,
The data structure also includes two Boolean to mark
if the face is a Gabriel face or a convex hull face.

The class \ccRefName\ is parameterized by a number type \ccc{NT}.
