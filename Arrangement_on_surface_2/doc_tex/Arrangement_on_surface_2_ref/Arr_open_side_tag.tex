\begin{ccRefClass}{Arr_open_side_tag}

\ccDefinition
%============

All the four types \ccc{Left_side_category},
\ccc{Right_side_category}, \ccc{Bottom_side_category},
and \ccc{Top_side_category} nested in any model of the
concept \ccc{ArrangementOpenBoundaryTraits} must be convertible
to \ccRefName, which derives from \ccc{Arr_oblivious_side_tag}. It
implies that some curves are expected to approach the left, right,
bottom, or top sides of the open boundary of the parameter
space. \ccRefName{} is an empty construct used for dispatching
functions based on type of curves that induce the arrangement.
