There are a number of mailing lists and email addresses that have been set 
up to communicate with various subsets of the \cgal\ community.  The 
addresses that should be of interest to \cgal\ developers are as follows:

\begin{description}
\item[\ccAnchor{mailto:cgal-develop@cs.uu.nl}{{\tt cgal-develop@cs.uu.nl}}] 
     \index{cgal_develop mailing list@{\tt cgal-develop} mailing list}
     This list is intended for discussing detailed matters 
     about implementation and development issues.  All developers should 
     subscribe to this list by sending a message to
     \ccAnchor{mailto:cgal-develop-request@cs.uu.nl}{cgal-develop-request@cs.uu.nl} 
     with the content ``\texttt{subscribe}''. 
     An \ccAnchor{http://postino.mpi-sb.mpg.de/cgi-bin/lyris.pl?enter=cgal-develop-l}{archive} 
     of this mailing list is maintained%
     \begin{ccTexOnly}
     at \nonlinkedpath|http://postino.mpi-sb.mpg.de/cgi-bin/lyris.pl?enter=cgal-develop-l|
     \end{ccTexOnly}.

\item[\ccAnchor{mailto:cgal-discuss-l@postino.mpi-sb.mpg.de}{{\tt cgal-discuss-l@postino.mpi-sb.mpg.de}}] 
     \index{cgal_discuss-l mailing list@{\tt cgal-discuss-l} mailing list}
     Users of \cgal\ post questions and discuss issues
     related to \cgal\ on this list.  All developers should subscribe to this 
     list so they can answer questions relevant to their packages and monitor 
     the input from the users provided here. 
     To subscribe to the list, send a message to
     \ccAnchor{mailto:cgal-discuss-l-request@postino.mpi-sb.mpg.de}%
     {\nonlinkedpath|cgal-discussion-l-request@postino.mpi-sb.mpg.de|}
     with the content ``\texttt{subscribe cgal-discuss-l}''.
     An \ccAnchor{http://postino.mpi-sb.mpg.de/cgi-bin/lyris.pl?enter=cgal-discuss-l}{archive} 
     of this mailing list is maintained%
     \begin{ccTexOnly}
     at \nonlinkedpath|http://postino.mpi-sb.mpg.de/cgi-bin/lyris.pl?enter=cgal-discuss-l|
     \end{ccTexOnly}.

\item[\ccAnchor{mailto:cgal-editor@cs.uu.nl}{{\tt cgal-editor@cs.uu.nl}}] 
     \index{cgal_editor address@{\tt cgal-editor} address}
     This is the email address to which you should
     send proposed design specifications (Chapter~\ref{chap:specification})
     for approval by the editorial 
     board (Section~\ref{sec:editorial_board}). Anyone can do that 
     at any time about any \cgal\ design issues. 

\item[\ccAnchor{mailto:cgal-submit@cs.uu.nl}{{\tt cgal-submit@cs.uu.nl}}] This is the address to which electronic
     \index{cgal_submit address@{\tt cgal-submit} address}
     submissions of packages, documentation, bug fixes, \etc should
     be sent (Section~\ref{sec:electronic_submission}).
\end{description}
