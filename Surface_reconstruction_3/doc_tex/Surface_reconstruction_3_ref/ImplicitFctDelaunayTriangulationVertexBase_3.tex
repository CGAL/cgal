\ccDefinition

% The section below is automatically generated. Do not edit!
%START-AUTO(\ccDefinition)

The \ccc{ImplicitFctDelaunayTriangulationVertexBase_3} concept defines the interface of the vertex class of the \ccc{ImplicitFctDelaunayTriangulation_3} concept. It provides the interface requested by the \ccc{Poisson_implicit_function} class.


Precondition: The geometric traits class 's \ccc{Point_3} type must be a model of \ccc{PointWithNormal_3}.
