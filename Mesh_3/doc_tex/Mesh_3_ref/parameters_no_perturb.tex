\ccDefinition
  
The function \ccRefName\ allows the user to tell mesh generation global functions
\ccc{make_mesh_3} and \ccc{refine_mesh_3} that no perturbation must be done.

\ccInclude{CGAL/refine_mesh_3.h}

\ccGlobalFunction{parameters::internal::Perturb parameters::no_perturb();}

\ccParameters

This function takes no parameter.



\ccExample

\begin{ccExampleCode}
// Mesh generation without perturbation
C3t3 c3t3 = make_mesh_3<c3t3>(domain, criteria, parameters::no_perturb());
\end{ccExampleCode}
