\begin{ccRefClass}{Random_points_on_sphere_d<Point_d>}

\ccDefinition

The class \ccRefName\ is an input iterator creating points uniformly
distributed on a sphere.

The generated points are computed using floating point arithmetic,
whatever the Kernel is, thus they are on the sphere only up to
rounding errors.

\ccIsModel
InputIterator \\

\ccCreationVariable{g}
\ccConstructor{Random_points_on_sphere_d(int dim,  double r, Random& rnd =
  default_random);}{%
  $g$ is an input iterator creating points of type \ccc{Point_d} uniformly
  distributed on a sphere in dimension $dim$ with radius $r$,
  i.e.~$|\ccc{*g}| == r$~. $2\cdot dim$ random numbers are needed from
  \ccc{rnd} for each point.
