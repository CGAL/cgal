
\authors Hans Tangelder  and Andreas Fabri

This package provides data structures and algorithms for exact and approximate distance browsing,
supporting

<UL>

<LI>both nearest and furthest neighbor searching,

<LI>both exact and approximate searching,

<LI>(approximate) range searching,

<LI>(approximate) \f$ k\f$-nearest and \f$ k\f$-furthest neighbor searching,

<LI>(approximate) incremental nearest and incremental furthest neighbor searching,

<LI>query items representing points and spatial objects.

</UL>

The spatial searching package consists of the following concepts and
classes that are described in the reference pages.
